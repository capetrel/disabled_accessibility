# Accessibility module

Accessibility module for Drupal 8.8 or 9.
It's a very simple module who add some classes attributes on the body.
There is no styling, the module only add classes on the HTML body tag.

## installation
  - copy module folder in /modules/custom/ directory
  - enable module in dashboard
  - go to module config page : /admin/config/system/disabled_accessibility/config and chose which item you want to active. with or without change You must save one time to regsiter module settings.
  - go to Block Layout and place tu accessibilty block where you want.
  - done

## styling
There just some css to align and style buttons, but you can override it. After you can make the accessibility style in your css's theme.
